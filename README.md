# EstatePrime - Real Estate Website

A modern, clean real estate website built with HTML, CSS, and JavaScript.

## 🌐 Live Demo
https://omushamotamashe-bot.github.io/estateprime/

## 📋 Features

- **Home Page** - Hero section with call-to-action
- **About Us** - Company information and stats
- **List a Home** - Submit property listings
- **Loan Help** - Request financing assistance
- **Contact** - Get in touch with us
- **Login / Register** - User authentication (demo)

## 🛠️ Technologies Used

- HTML5
- CSS3 (Custom styling)
- JavaScript
- Font Awesome (Icons)
- Google Fonts (Inter)
